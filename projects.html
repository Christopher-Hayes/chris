Animated Typing in JS

As a fun side project where I tackled the task of replicating the Portal 1 and 2 credits scenes with HTML/CSS/JS. The big challenge for this project was animating the text in a way that it looked realistic and matched the speed of the lyrics. This was accomplished with animations on span tags and developing tools to calculate timing.

Cubefield with Three.JS

This another fun side project where I attempted to bring software from another medium to HTML/CSS/JS. The original cubefield was a flash game. This project recreates the game with the help of Three.JS The overall game and the level colors match the original game, but I took some liberty with the game options to make it more interesting.
